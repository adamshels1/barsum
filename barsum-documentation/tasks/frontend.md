Стек: Next.js 16 · React 19 · TypeScript · Tailwind CSS v4 · Zustand · TanStack Query · Playwright

## ИНФРАСТРУКТУРА

### TASK-F-01 — Базовая настройка
- [x] Route groups: `(child)`, `(parent)`, `(expert)`, `(admin)`
- [x] Middleware (`src/proxy.ts`) — защита роутов по роли
  - `/child/*` → только `role: child`
  - `/parent/*` → только `role: parent`; `/parent/onboarding` доступна сразу после регистрации
  - `/expert/*` → только `role: expert`; если `expertStatus: new|review` → редирект на `/expert/onboarding`
  - `/admin/*` → только `role: admin`
  - Без токена → редирект на `/`
- [x] Layout для каждой группы
- [x] `src/lib/api/` — модули по фичам: `auth.ts`, `children.ts`, `challenges.ts`, `sessions.ts`, `rewards.ts`, `coins.ts`, `payments.ts`, `experts.ts`, `admin.ts`
- [x] `src/types/` — TypeScript типы для всех сущностей
- [x] `src/hooks/` — базовые хуки: `useCurrentUser`, `useChildBalance`, `useParentBalance`
- [x] Playwright: тест что middleware редиректит неавторизованного

### TASK-F-02 — UI Kit (базовые компоненты)
