# Barsum UI Improvements — Задачи

## РЕБЁНОК

### TASK-UI-01 — Фото мечты как фон карточки 🟢
**Референс:** Monarch Financial (goal card с фото), Revolut (3D-иллюстрация мечты)
**Страницы:** `/child/home`, `/child/shop`

- [x] На `/child/home`: если у мечты есть `photoUrl` — показывать его как фон виджета с тёмным градиентом поверх
- [x] Текст (название, прогресс) — белый, поверх фото
- [x] Если фото нет — текущий фиолетовый градиент остаётся
- [x] На `/child/shop` вкладка «Мечта»: аналогично — фото как фон карточки активной мечты
- [x] Fallback: если фото не загрузилось — фиолетовый градиент

- [x] Анимация обновления числа монет при начислении (счётчик прокручивается вверх)

### TASK-UI-06 — Мотивирующие подписи на прогресс мечты 🟢
**Референс:** Mint ("You go, goal-getter!"), Revolut ("+Fund" на карточке)
**Страницы:** `/child/home`, `/child/shop`

- [x] Под прогресс-баром мечты: динамическая фраза вместо просто чисел
  - 0–10%: «Только начинаем! Читай каждый день 🚀»
  - 11–49%: «Уже {N} монет! До мечты осталось {M} 💪»
  - 50–89%: «Больше половины! {dream.name} уже близко ✨»
  - 90–99%: «Почти! Ещё чуть-чуть и {dream.name} твой! 🔥»
  - 100%: «Мечта достигнута! 🎉»
- [x] Кнопка «Внести монеты» прямо на виджете home-экрана (сейчас она только в shop)

**Страницы:** `/child/shop`
